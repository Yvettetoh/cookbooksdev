# Examples

## Config

```sh
mc config host add [alias] [url] [accesskey] [secretkey]
```

## List

```sh
mc ls [repo]/[bucket]
```

## Remove

```sh
mc rm -r --force [repo]/[bucket]
```

## Make Bucket

```sh
mc mb [repo]/[bucket]
```

## Policy

```sh
mc policy [public|upload|download|list] [repo]/[bucket]
```
