# Linux

## Installation

### Homebrew

```sh
brew install supervisor
```

### APT

```sh
sudo apt update
sudo apt -y install supervisor
```

### YUM

```sh
yum check-update

# Repo: EPEL
sudo yum -y install supervisor
```

### APK

```sh
sudo apk update
sudo apk add supervisor
```

## Commands

```sh
supervisord --help
```
