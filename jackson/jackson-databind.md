# Jackson Data-binding

## Library

### Apache Maven

#### Project Object Model (POM)

```xml
<properties>
  <!-- ... -->
  <jackson.version>2.12.3</jackson.version>
</properties>

<dependencies>
  <!-- ... -->
  <dependency>
    <groupId>com.fasterxml.jackson.core</groupId>
    <artifactId>jackson-databind</artifactId>
    <version>${jackson.version}</version>
  </dependency>
</dependencies>
```
