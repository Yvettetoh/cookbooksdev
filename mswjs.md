# Mock Service Worker (MSW)

**Keywords:** Mocking

## Links

- [Code Repository](https://github.com/mswjs/msw)
- [Main Website](https://mswjs.io/)
- [Examples](https://github.com/mswjs/examples/tree/master/examples)

## Library

```sh
# Using NPM
npm install msw --save-dev

# Using Yarn
yarn add msw --dev
```

## CLI

### Commands

```sh
npx msw -h
yarn dlx msw -h
```

## Usage

```sh
#
npx msw init ./public
```
