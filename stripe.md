# Stripe

## Links

- [Main Website](https://stripe.com/)

## CLI

### Installation

#### Homebrew

```sh
brew tap stripe/stripe-cli
brew install stripe
```

### Commands

```sh
stripe help
```

### Usage

```sh
#
stripe login

#
stripe customers create
```

## Python Library

### Installation

#### pip

```sh
pip3 install -U stripe
```

### Usage

```sh
python3 ./manage.py migrate
```
