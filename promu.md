# Prometheus Utility Tool

## CLI

### Installation

#### Darwin Binary

```sh
# Darwin
curl \
  -L \
  'https://github.com/prometheus/promu/releases/download/v0.5.0/promu-0.5.0.darwin-amd64.tar.gz' | \
    tar -xzC /usr/local/bin --strip-components 1 promu-0.5.0.darwin-amd64/promu
```

### Commands

```sh
promu -h
```
