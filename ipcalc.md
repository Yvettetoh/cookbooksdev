# ipcalc

## CLI

### Installation

#### Homebrew

```sh
brew install ipcalc
```

#### APT

```sh
sudp apt update
sudo apt -y install ipcalc
```

#### YUM

```sh
sudo yum check-update
sudo yum -y install ipcalc
```

### Commands

```sh
ipcalc -h
```

### Usage

```sh
#
ipcalc [1.2.3.4] [255.255.255.0]

#
ipcalc [1.2.3.4]/[24]

#
ipcalc -b [1.2.3.4]/[30]

#
ipcalc -n [1.2.3.4]/[24]

#
ipcalc -g [1.2.3.4]
```
