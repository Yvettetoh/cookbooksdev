# Apache Groovy

## CLI

### Installation

#### Homebrew

```sh
brew install groovy
```

### Environment

For Bash or Zsh, put something like this in your `$HOME/.bashrc` or `$HOME/.zshrc`:

```sh
# Groovy
export GROOVY_HOME='/usr/local/opt/groovy/libexec'
```

```sh
sudo su - "$USER"
```

### Commands

```sh
groovy -h
```

### Usage

```sh
# Version
groovy -v 2>&1 | tail -1
```
