# Apache JMeter

https://github.com/SantimosoTFM/tfm-loadtesting/tree/master/jmeter

## CLI

### Installation

#### Homebrew

```sh
brew install jmeter --with-plugins
```

### Uninstall

#### Homebrew

```sh
brew uninstall jmeter
```

### Plugins

#### Homebrew

```sh
open $(brew --prefix jmeter)/libexec/lib/ext/
```
