## CLI

### Commands

```sh
# Using NPM
npx typeorm -h

# Using Yarn
yarn dlx typeorm -h
```

### Usage

```sh
#
yarn dlx typeorm migration:generate -n

#
yarn dlx typeorm migration:create -n init

#
yarn dlx typeorm migration:run

#
yarn dlx typeorm migration:revert
```
