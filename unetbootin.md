# UNetbootin

## App

### Installation

#### Homebrew

```sh
brew install --cask unetbootin
```

### Docs

#### Windows

1. UNetbootin needs permissions to write to your USB drive. Type your password to allow this.
   - Type your password
2. Check "Diskimage"
   - Select "ISO"
   - Choose `WinXX_x64.iso` file
3. Type "USB Drive"
   - Select correct drive
   - Ok

### Uninstall

```sh
# Homebrew
brew uninstall --cask unetbootin
```
