# saml2aws

Login and retrieve AWS temporary credentials using a SAML IDP

## Links

- [Code Repository](https://github.com/Versent/saml2aws)

## CLI

### Installation

#### Homebrew

```sh
brew install saml2aws
```

#### Chocolatey

```sh
choco install saml2aws
```

### Commands

```sh
saml2aws --help
```

### Usage

```sh
#
saml2aws login --force
```
