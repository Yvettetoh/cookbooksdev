## CLI

### Installation

#### Homebrew

```sh
brew install fluxctl
```

#### Snappy

```sh
sudo snap install fluxctl --edge
```

### Commands

```sh
fluxctl -h
```

### Usage

```sh
# List Workloads
fluxctl list-workloads --k8s-fwd-ns flux
```
