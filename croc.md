# croc

**Keywords:** Transfer Files P2P

## Links

- [Code Repository](https://github.com/schollz/croc)

## CLI

### Installation

#### Homebrew

```sh
brew install croc
```

#### Chocolatey

```sh
choco install croc
```

### Commands

```sh
croc -h
```

### Usage

```sh
#
croc send [./my-file.ext]

#
croc send \
  --code [code-phrase] \
  [./my-file.ext]
```
