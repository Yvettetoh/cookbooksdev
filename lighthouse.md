# Lighthouse

## CLI

### Installation

#### Homebrew

```sh
brew install lighthouse
```

#### NPM

```sh
npm install lighthouse -g
```

### Commands

```sh
lighthouse --help
```

### Usage

```sh
#
lighthouse http://example.com \
  -G

#
lighthouse http://example.com \
  -A

#
lighthouse http://example.com \
  -GA

#
lighthouse http://localhost:3000/docs/ \
  --chrome-flags='--ignore-certificate-errors' \
  --view
```
