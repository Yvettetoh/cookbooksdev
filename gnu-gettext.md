# GNU Gettext

## CLI

### Installation

#### Homebrew

```sh
brew install gettext
```

#### APT

```sh
sudo apt update
sudo apt -y install gettext libgettextpo-dev
```

#### YUM

```sh
sudo yum check update
sudo yum -y install gettext-devel
```

#### APK

```sh
apk update
apk add gettext gettext-dev
```

### Tips

#### Visual Studio Code

```sh
code --install-extension mrorz.language-gettext
```
