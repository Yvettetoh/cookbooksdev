## CLI

### Installation

#### Homebrew

```sh
brew tap spotinst/tap
brew install spotctl
```

### Commands

```sh
spotctl -h
```

### Environments

```sh
#
export SPOTINST_TOKEN='[token]'
export SPOTINST_ACCOUNT='[account]'
```

### Usage

```sh
#
spotctl configure

#
cat ~/.spotinst/credentials
```
