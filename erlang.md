# Erlang

## CLI

### Installation

#### Homebrew

```sh
brew install erlang
```

#### APT

```sh
sudo apt update
sudo DEBIAN_FRONTEND=noninteractive apt -y install erlang
```

#### APT

```sh
yum check-update

# Repo: EPEL
sudo yum -y install erlang
```

### Commands

```sh
erl -man
```

### Usage

```sh
# REPL
erl
```
