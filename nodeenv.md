# Node.js Virtual Environment (nodeenv)

## Links

- [Code Repository](https://github.com/ekalinin/nodeenv)

## CLI

### Installation

#### Homebrew

```sh
brew install nodeenv
```

### Commands

```sh
nodeenv -h
```

### Usage

```sh
#
nodeenv ./.env

#
source ./.env/bin/activate

#
rm -fR ./.env
```
