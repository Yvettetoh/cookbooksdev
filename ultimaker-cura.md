## App

### Installation

#### Homebrew

```sh
brew cask install ultimaker-cura
```

### Configuration

#### Creality Ender 3 v2

1. Settings -> Printer -> Add Printer
2. Add a non-networked printer
3. Creality3D -> Choose "Creality Ender-3"
   - Change "Printer Name" to "Creality Ender-3 v2"
   - Click Add
4. Machine Settings
   - Printer Tab
     - X (Width): 220
     - Y (Depth): 220
     - Next
