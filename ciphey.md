# Ciphey

## Links

- [Code Repository](https://github.com/Ciphey/Ciphey)

## CLI

### Installation

#### Homebrew

```sh
brew install ciphey
```

### Commands

```sh
ciphey --help
```

### Usage

```sh
# File Input
ciphey -f ./encrypted.txt

# Unqualified input
ciphey -- '[encrypted]'

# Normal way
ciphey -t '[encrypted]'
```
