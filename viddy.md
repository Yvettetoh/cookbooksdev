# Viddy

## Links

- [Code Repository](https://github.com/sachaos/viddy)

## CLI

### Installation

#### Homebrew

```sh
brew tap sachaos/tap
brew instal viddy
```

#### Linux Binary

```sh
curl \
  -L \
  'https://github.com/sachaos/viddy/releases/download/v0.1.6/viddy_0.1.6_Linux_x86_64.tar.gz' | \
    tar -xzC /usr/local/bin viddy
```

### Commands

```sh
viddy -h
```

### Usage

```sh
#
viddy -n 1 [command]
```
