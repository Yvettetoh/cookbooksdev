# libxml2

## Library

### Installation

#### Homebrew

```sh
brew install libxml2
```

#### APT

```sh
sudo apt update
sudo apt -y install libxml2-dev
```

#### YUM

```sh
yum check-update
sudo yum -y install libxml2-devel
```

#### Zypper

```sh
sudo zypper refresh
sudo zypper install -y libxml2-devel
```
